Project based on following Youtube-Videos (#13 until #24):
https://www.youtube.com/watch?v=cNU5vlbq7k4&list=PLsyeobzWxl7rRyGcqgZ3MP5pWGPwUvprI&index=13

Technology used:
+ Java
+ Spring (JPA, Spring Cloud Netflix - Eureka Server, Cloud OpenFeign, Cloud Gateway, Thymeleaf)
+ PostgresSQL data base 
+ Postman (for communication http GET, POST)
+ HTML, CSS

Execution:
Run each service separately in the IDE, first service-registry, then api-gateway, finally quiz-question-service.
